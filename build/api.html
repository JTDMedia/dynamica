Dynamica API Documentation

Welcome to the official API documentation for Dynamica, a fast and free-to-use proxy service. This API allows you to fetch the HTML content of any URL without compromising your privacy. Below are the details on how to use the API.

API Endpoint

The main endpoint for the API is:

POST /proxy

To use the API, send a POST request to the `/proxy` endpoint with the following parameters:

  • pageURL: The URL that you want to proxy (e.g., https://example.com).

Request Example

Hereâ€™s an example of how to send a request to the API using JavaScript (fetch):


fetch('https://brilliant-baklava.netlify.app/proxy', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({ pageURL: 'https://example.com' })
})
.then(response => response.text())
.then(html => {
    console.log(html);  // The HTML content of the requested page
});

API Authentication

Currently, there is no authentication required to use the API. However, in the future, an API key will be necessary to authenticate requests. Limits on the number of requests will also be enforced. Stay tuned for updates!
